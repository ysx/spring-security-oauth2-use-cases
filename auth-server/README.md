# auth-server （认证&授权中心）
> 使用Spring Boot框架。统一认证中心。

功能描述：
- 客户端模式认证
- 密码模式认证
- 封装的登录接口认证（密码模式）
- 验证token
- 获取当前登录用户
- 退出登录
